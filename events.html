📋 Google Calendar Setup Instructions

To activate the live calendar, complete these steps:

  1. Create a Google Calendar — Go to calendar.google.com, click the "+" next to "Other calendars", and name it "NLRC Events".
  2. Make it public — Calendar Settings → "NLRC Events" → Access permissions → check "Make available to public".
  3. Copy your Calendar ID — Settings → "Integrate calendar" → copy the Calendar ID (looks like abc123@group.calendar.google.com).
  4. Share the ID with your developer — Replace YOUR_CALENDAR_ID_HERE in events.html with your actual Calendar ID.
  5. Share calendar access with admin — Add your admin team's Google accounts with "Make changes to events" permission so they can add/edit events.

✅ Once configured, the calendar updates automatically — no code changes needed to add/edit events!
